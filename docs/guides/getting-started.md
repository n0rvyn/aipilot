# 快速开始指南

欢迎使用 AIPilot！本指南将帮助您快速上手这个强大的 AI 助手插件。

## 📦 安装

### 方法一：通过 Obsidian 社区插件（推荐）

1. 打开 Obsidian
2. 进入 `设置` → `社区插件`
3. 点击 `浏览`
4. 搜索 "AIPilot"
5. 点击 `安装`
6. 安装完成后，点击 `启用`

### 方法二：手动安装

1. 从 [GitHub Releases](https://github.com/norvyn/aipilot/releases) 下载最新版本
2. 下载以下文件：
   - `main.js`
   - `manifest.json`
   - `styles.css`
3. 在你的 Obsidian 库中创建文件夹：`.obsidian/plugins/aipilot/`
4. 将下载的文件放入该文件夹
5. 重启 Obsidian
6. 在 `设置` → `社区插件` 中启用 AIPilot

## ⚙️ 基础配置

### 第一步：配置 API 密钥

1. 打开 `设置` → `AIPilot`
2. 选择你的 AI 提供商：
   - **OpenAI**: 需要 OpenAI API Key
   - **智谱 AI**: 需要智谱 API Key
   - **Groq**: 需要 Groq API Key
   - **自定义**: 任何兼容 OpenAI API 的服务

3. 输入你的 API 密钥

#### 如何获取 API 密钥？

**OpenAI**:
1. 访问 [platform.openai.com](https://platform.openai.com)
2. 注册/登录账号
3. 进入 [API Keys](https://platform.openai.com/api-keys)
4. 点击 "Create new secret key"
5. 复制密钥（只会显示一次！）

**智谱 AI**:
1. 访问 [open.bigmodel.cn](https://open.bigmodel.cn)
2. 注册/登录账号
3. 进入控制台获取 API Key

**Groq**:
1. 访问 [console.groq.com](https://console.groq.com)
2. 注册/登录账号
3. 创建 API Key

### 第二步：选择模型

根据你的需求选择合适的模型：

| 模型 | 特点 | 适用场景 |
|------|------|----------|
| GPT-3.5-Turbo | 快速、便宜 | 日常对话、简单任务 |
| GPT-4 | 强大、准确 | 复杂推理、专业任务 |
| GLM-4 | 中文优化 | 中文写作、中文对话 |
| Llama 3 | 快速、免费 | 基础任务、高频使用 |

### 第三步：测试连接

1. 点击侧边栏的 AIPilot 图标 📱
2. 在聊天框输入 "你好"
3. 如果收到回复，说明配置成功！

## 🎯 核心功能快速上手

### 1. AI 聊天

**打开聊天界面**:
- 点击左侧边栏的 AIPilot 图标
- 或使用命令面板（Ctrl/Cmd + P）搜索 "AIPilot: Open Chat"

**基础对话**:
```
你：如何提高写作水平？
AI：提高写作水平可以从以下几个方面入手...
```

**插入回复到笔记**:
- 点击消息右上角的 `插入` 按钮
- 内容会插入到当前活动编辑器

### 2. 文本润色

**使用步骤**:
1. 在编辑器中选中需要润色的文本
2. 打开命令面板（Ctrl/Cmd + P）
3. 输入 "Polish" 选择 `AIPilot: Polish Text`
4. 查看 AI 的修改建议（标注了删除和新增）
5. 点击 `应用修改` 接受更改

**可视化差异**:
- 🔴 红色删除线：被删除的内容
- 🟣 紫色高亮：新增的内容

**示例**:

原文:
```
这个方法很好用，可以帮助我们解决问题。
```

润色后:
```
<del>这个</del><mark>该</mark>方法<del>很好用</del><mark>高效实用</mark>，<del>可以</del><mark>能够</mark>帮助我们<del>解决问题</del><mark>快速解决实际问题</mark>。
```

### 3. 快捷功能按钮

聊天界面顶部有多个功能按钮：

| 图标 | 功能 | 说明 |
|------|------|------|
| 📝 | 组织 | 整理和结构化内容 |
| ✍️ | 润色 | 改进文字表达 |
| 🎨 | 生成 | 创意写作 |
| 💬 | 对话 | 自然对话 |
| 📚 | 摘要 | 内容总结 |
| ⭐ | 自定义 | 你的自定义功能 |

**使用方法**:
1. 点击功能按钮
2. 输入内容
3. AI 会根据功能预设的提示词处理

### 4. 引用笔记内容

**在对话中使用知识库**:

1. **手动引用**:
   ```
   参考 [[我的笔记]]，请告诉我...
   ```

2. **启用 RAG 模式**:
   - 点击聊天界面的 "启用知识库" 开关
   - AI 会自动检索相关笔记作为上下文

**RAG 模式优势**:
- ✅ 基于你的笔记回答
- ✅ 减少 AI 幻觉
- ✅ 可追溯信息来源

### 5. 聊天历史

**保存对话**:
1. 点击聊天界面的 `历史` 按钮
2. 选择 `保存当前对话`
3. 对话会保存为 Markdown 文件

**查看历史**:
1. 点击 `历史` 按钮
2. 从列表中选择历史对话
3. 点击恢复到聊天界面

**配置保存位置**:
- 设置 → AIPilot → 聊天历史路径
- 默认：`AI Chats/`

## 🎨 自定义功能

### 创建自定义功能

1. 打开 `设置` → `AIPilot` → `自定义功能`
2. 点击 `添加功能`
3. 填写以下信息：

   - **名称**: 功能显示名称（如 "代码审查"）
   - **图标**: 选择一个图标（如 "code"）
   - **提示词**: 定义 AI 的行为

4. 点击 `保存`

### 示例：代码审查功能

```
名称: 代码审查
图标: code
提示词:
你是一位资深的代码审查专家。请仔细审查用户提供的代码，并提供：
1. 潜在的 bug 和问题
2. 性能优化建议
3. 代码风格和最佳实践建议
4. 安全性问题（如果有）

请用友好、建设性的语气给出反馈。
```

### 示例：学术润色功能

```
名称: 学术润色
图标: graduation-cap
提示词:
你是一位学术写作专家。请帮助用户改进学术论文的表达，确保：
1. 使用正式、学术化的语言
2. 表达清晰、逻辑严密
3. 符合学术规范
4. 保持原意不变

请只输出改进后的文本，不要添加额外解释。
```

## 💡 使用技巧

### 技巧 1：使用上下文

将相关内容复制到聊天框，帮助 AI 更好理解：

```
我正在写一篇关于【主题】的文章。以下是我的大纲：

1. 引言
2. 主要论点
3. 结论

请帮我扩展第二部分。
```

### 技巧 2：迭代改进

不满意结果？继续对话：

```
你：请帮我写一段开场白
AI：[生成内容]
你：能否更简洁一些？
AI：[改进版本]
你：很好！再加入一个例子
AI：[最终版本]
```

### 技巧 3：明确要求

越具体的要求，结果越好：

❌ 差：
```
写点东西
```

✅ 好：
```
写一篇 500 字的科普文章，介绍量子计算，
面向高中生，使用通俗易懂的比喻。
```

### 技巧 4：使用分隔符

处理长文本时使用分隔符：

```
请润色以下内容：

---

[你的长文本]

---

要求：
- 保持原意
- 提高可读性
- 修正语法错误
```

## 🔧 常见问题

### Q: API 调用失败怎么办？

**A**: 检查以下几点：
1. API 密钥是否正确
2. 账户余额是否充足
3. 网络连接是否正常
4. 是否需要配置代理

### Q: 响应速度慢？

**A**: 可以尝试：
1. 切换到更快的模型（如 GPT-3.5、GLM-3-Turbo）
2. 减少上下文长度
3. 关闭 RAG 功能（如果不需要）
4. 使用 Groq（速度极快）

### Q: 如何保护隐私？

**A**: 
1. 不要在提示词中包含敏感信息
2. 使用本地模型（未来支持）
3. 定期删除聊天历史
4. API 密钥已加密存储

### Q: 中文支持如何？

**A**: 
- GPT-4：优秀
- GPT-3.5：良好
- GLM-4：优秀（中文优化）
- Llama 3：良好
